Breadth-First Search (BFS) : This algorithm follows the Queue’s 

structure that is     First-In-First-Out(FIFO). There can be 2 possible 
shortest paths:  

1. PATH : A - B - E - D - C - A  , COST : 4 + 2 + 1 + 3 + 1 = 11 
2. PATH : A - C - E - D - B - A  , COST : 1 + 2 + 1 + 5 + 4 = 13 

 

● Depth-First Search (DFS) : This algorithm follows the Stack structure 

that is Last-In-First-Out(LIFO). There can be 2 possible shortest 
paths:  

1. PATH : A - C - D - E - B - A  , COST : 1 + 3 + 1 + 2 + 4 = 11 
2. PATH : A - B - D - E - C - A  , COST : 4 + 5 + 1 + 2 + 1 = 13
