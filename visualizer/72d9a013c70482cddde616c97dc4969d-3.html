Program Description  

 

Story 

The virtual maze world is a two-dimensional grid of squares, 

some are empty. There is a Start and a Goal state the same as a 

Pacman game where the pac-man finds a way to reach its food 

and take the optimal path while avoiding enemies which in our 

case will be walls. 

Objective 

● Reach the goal-state. 

● Explore the shortest path possible while evaluating the 

expanded nodes and calculating the path cost from start 

state to goal state. 

Settings 

● The maze provided in .txt files (bigMaze.txt and 

mediumMaze.txt) 

● “%” indicates the walls of the maze 

● “.” dot indicates the path followed by the algorithm to reach the 

goal state  

● “P” indicates the start state
